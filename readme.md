# Resume Website (HTML)

## Description

This project is a personal CV/resume webpage built using HTML. It displays personal details, education, and skills in a structured format.

LIVE DEMO:-https://shashikantmulimani076.github.io/my-cv/

## Features

* Personal information section
* Education details
* Skills listing
* Simple and readable layout

## Technologies Used

* HTML5

## Usage

Open the HTML file in a browser to view the resume.
